# Gushwork Frontend Assessment

A responsive landing page built using **pure HTML, CSS, and JavaScript** as part of a frontend assessment.

The project focuses on **clean UI implementation, responsive design, and component-based layout**, following the design requirements provided in the assessment.

## Tech Stack

* HTML5
* CSS3 (Responsive Design)
* Vanilla JavaScript
* Git & GitHub
* GitHub Pages (for deployment)

---

## Features

* Fully responsive layout for **desktop, tablet, and mobile**
* Clean and modular **component-based UI sections**
* Smooth **horizontal scroll cards**
* Responsive **grid-based layouts**
* Interactive **tabs and accordion components**
* Optimized for **modern browsers**

---

## Sections Implemented

* Hero Section
* Trust Bar
* Features Section
* Applications (Horizontal Scroll Cards)
* Manufacturing Process Tabs
* Testimonials (Responsive Scroll Cards)
* Portfolio Section
* FAQ Accordion
* Contact / CTA Section
* Footer

---

## Responsive Behaviour

The layout adapts across multiple screen sizes:

* Desktop (>800px)
* Tablet (≤800px)
* Mobile (≤360px)

Each section adjusts using **CSS Grid, Flexbox, and media queries** to ensure consistent user experience.
